# Azure Stock Anomaly Detection

## Project Overview

This project is a cloud-native stock anomaly detection platform built on Microsoft Azure. It automatically detects stock price anomalies, processes events, and deploys the application on Azure Kubernetes Service (AKS).

## Technologies Used

- Azure Functions
- Azure Event Hub
- Azure Kubernetes Service (AKS)
- Docker
- Helm
- Terraform
- GitHub Actions
- Jenkins
- Azure Container Registry (ACR)
- Azure Key Vault
- RBAC
- Application Insights
- Azure Monitor
- Grafana
- Loki
- Python

## Features

- Event-driven architecture
- Infrastructure as Code using Terraform
- CI/CD with GitHub Actions and Jenkins
- Containerized with Docker
- Kubernetes deployment using Helm
- Secure secret management with Azure Key Vault
- Monitoring using Grafana and Loki
- Logging with Application Insights
